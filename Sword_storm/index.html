SWORD STORM

  • Originality: You get to dodge falling swords with UNLIMITED LIVES!!
  • Technical Merit: It is very simple. It uses the Phaser 2 game framework and uses only 2 keyboard keys.
  • Prototype Postmortem: I learnt how to make a very simple game using the Phaser 2 javascript framework. I also learnt what prototyping means and why it si very important. The hardest part was getting used to the functions that the framework provides. Since this is my first game using Phaser 2 it was a challenge trying to implement certain features in the game.

  • Prototype Closest Other Game: The game is kind of unique. It is not unique in its genre but there are very few games that are similar to this game.
  • High Concept: The is about controlling a small man to avoid the falling swords.
  • Theme: I feel themes of this game are fast paced, dark and death.
  • Mandated Variety: The input used is the keyboard. The user plays the game using only two keys on the keyboard. The genre is Platform and falling objects.
  • Prototype Goal: Some of the game mechanics that are being evaluated are area movement, instant killing and avoidng killable objects
  • Player Experience Goals: The player should be experience a fast paced environment by the falling swords and progressive music.
  • Gameplay: The gameplay is very simple. The input of the game is using the keybaord. The core mechanic is moving left and right. player moves left or right by clicking on the left or right arrow key repectively. Essentailly the full game will include the player having 3 lives or 3 chances and if the player looses all 3 lives, it is game over and restarts the game for another shot. Over time as the player survives longer, the swords begin to fall a little bit faster to make the game for challenging. Living forever in a game is not fun so this gives the player a challenge to test how fast one's reflexes can be.
  • Strategies: A strategy that can be used in playing this game would be to not make your movements extremely random but notice the trends as to which areas of the screen the swords fall more often and avoid those areas.
  • Story/Setting/Premise: The man you play in the game is a daredevil that has challenged the world that if the world rained swords, he would never die. To prove this, he locks himself in a room with no exists and swords falling from random parts of the ceiling. The tokens do not mean anything more than a sword ready to kill him. The tension of proving to the world that you can survive a rainfall of swords lies in the hands of the player controling the man. Given three chances, can you survive??
  • Target Audience: Users above the age of 8 can play this game.
  • Play Time: The game lasts depending on how good the user is. It could last from a few seconds to forever.
